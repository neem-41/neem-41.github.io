Decision Tree

There are a lot of supervised machine learning algorithms that can be used for classification. Likewise, decision tree is a supervised machine learning algorithm. The unique thing about this algorithm is that it is used to classify data that requires conditional control statement.In simpler words it can classify data which have different cases. It is also used in regression models in some cases.
