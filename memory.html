Memory Mangement

In the project, I simulated a virtual memory space to mimick memory managements done by operating systems. This project is not a user based project but rather an program that goes through different steps to implement memory management done by operating systems in a virtual space.

The goal of this project was to understand different ideas like TLB table, page table, physical memory space and so on. This program hinged on the idea of undrestanding what goes on a computer on a system level. For the first part of the program, I had to figure out how to convert logical addresses into physical addresses. Then using those values I had to implement the various structure mentioned before to manage memory. While the project itself cannot be shown in a impressive manner, the fact that the program runs correctly is a great sign of my understanding of the material.

From this project, I have learned how memory is managed in a computer systems. On top of that I have learned concepts like page replacement and not I am able to implement them as well. I now know how hardware interacts with softwares and have also learned possible ways that data could get corrupted if some functionality of hardware did not exist.
